# React CV
This is my CV made using Vite React.

To view the demo : [Click here](https://melcv.netlify.app)

# How to get started
```
# Install dependencies
npm install

# Update/fix dependencies
npm audit fix

# Run
npm run dev 
```

# Tools
- Build Framework : Vite
- Library : React
- CSS : Sass
- Icons : Bootstrap icons
